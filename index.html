Latest Project

Another World Remake

Another World Remake is a 3D game developed with Unreal Engine as a first-year group project. It recreates the prison section of Eric Chahi’s classic Another World in 3D. Players, trapped by an alien civilization, must escape with another prisoner using a weapon that fires lasers and creates energy shields. The game blends third-person action and puzzle-solving while staying true to the original experience.
